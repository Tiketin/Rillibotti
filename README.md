# Rillibotti
Rillibotti on Discord-palvelimille tarkoitettu botti.

# Toiminnallisuus
Rillibotti osaa vaihtaa käyttäjän roolia .env muuttujassa annetun listan perusteella, sekä hakea F1-tuloksia rajapinnasta (https://api.jolpi.ca/ergast)

# Kehitys
Luo .env tiedosta ja täytä tarvittavat tiedot
riippuvuudet -> npm install
ajo -> npm start
testit -> npm test
